The Web Stack

The web stack are a collection of technologies use to create web pages.

 - - - - - - - - 
| Javascript    |   Client side scripting
 - - - - - - - - 
| CSS           |   Styling
 - - - - - - - - 
| HTML          |   Markup (page structure)
 - - - - - - - - 
| PHP           |   Server-side scripting
 - - - - - - - - 
| MySQL         |   Database layer
 - - - - - - - -

The technologies above HTML are client-side and are downloaded and executed by the browser. Those below HTML are server-side.

Much link common web site development, it is a good idea to keep your different technologies as separate as possible.

Development

It is possible, and is usually the case, that PHP will be developed on one system and deployed on another.

For example: you develop on a Mac. You stage to a Windows server attached to your local network and then deploy to a production machine running Ubuntu on a remote, shared host.

While this doesn't usually cause problems you have to be mindful of each successive step. Directory structures, file permissions, software versions, installed libraries and modules, database credentials, file paths, etc.
