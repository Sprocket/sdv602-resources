PHP Variables

PHP is a loosely typed language - a variable can contain any variable type and doesn't need to be declared when setting it. The type can be changed simply by reassigning a new value. You can cast variable; that is, change the type.

Variables do not need to be declared before being set (encouraged in OOP)

Variable Types

Booleans

the values TRUE or FALSE (case insensetive)

Integers

Number without a decimal, including negative numbers maximum values are platform dependent, usually about 2 Billion can be specified in decimal, hexadecimal, octal or binary
e.g.1, -1, 1024, 0

Floating point numbers

Numbers with decimal places, also known as "floats", "doubles", or "real numbers"
e.g. 1.234, 1.2e3, 7E-10

Strings

A series of characters quoted within single or double quote marks. Special characters within single quoted strings are not expanded. These include variables and whitespace. Symbols within double quotes will be converted to their values before returning the string to the browser.
e.g. 'Hello, World', "antidisestablishmentarialism", "123"

Arrays

A map of associated values and keys In other languages these are known as list, hash table, dictionary, collection, stack, etc. We create an array using the array keyword. An array may contain other arrays (a 'multi-dimensional array'). Keys have to be integers or strings. A value can be of any type, including variables.

// a one dimensional array
array( 'apple', 'banana', 'cantelop');
// a associative array 
array( "foo" => "bar"); 
// a multidimensional array (an array of arrays)
array( 'foobar' => array( "foo" => "bar") );

Objects

An object is a collection of functions and variables ('methods' and 'properties').

Resources

An external resource, such as a database connectino, a directory listing, an image file, etc.

NULL

A variable that exists that has no value or:

  • it has been assigned the constant NULL
  • it has not been set to any value yet
  • it has been unset()
