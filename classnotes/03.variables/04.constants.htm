Constants

Constants are just like variables but the value cannot be changed.

You use them just like variables but they don't have a $ before them. They are usually given all uppercase.

Constants exist in the global scope and can be accessed in any of your scripts. You define contstants using the define function, passing it the constant name and value.

define('USER', 'John Doe');
echo USER; // John Doe

You can test for a specific using defined

if ( defined('USER'))
echo 'Hello'. USER;
