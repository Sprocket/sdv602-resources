Working with Arrays

Foreach

Foreach is a way of looping through an array and performing an operation on each array item. It accepts an array and the name of new variable to use within the operation:

$fruits = array('apples','bananas','grapes');

foreach ( $fruits as $fruit ) {
	echo "{$fruit} are tasty<br />";
}
// apples are tasty
// bananas are tasty
// grapes are tasty
