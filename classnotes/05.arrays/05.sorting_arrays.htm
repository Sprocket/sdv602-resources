Sorting Arrays

There are several functions built in to PHP that allow you to rearrange arrays in a number of ways.

Sort

$letters = array('z','y','a','n','b','x','c');
$numbers = array(1, 7, 3, 85, -3, 1);
sort($numbers);
print_r($numbers);
sort($letters);
print_r($leters);

ksort

This will sort an assoc. array by keys

$people = array(
	'Sam'	=> 17,
	'Ben'	=> 22,
	'Sue'	=> 28,
	'Trish'	=> 19
);
ksort($person);
print_r($person);
Array
(
    [Ben] => 22
    [Sam] => 17
    [Sue] => 28

)

asort

This will sort an array by value

asort($person);
print_r($person);
Array
(
    [Sam] => 17
    [Trish] => 19
    [Ben] => 22
    [Sue] => 28
)
