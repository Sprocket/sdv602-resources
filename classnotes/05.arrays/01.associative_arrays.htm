Associative arrays

You can define your own keys within arrays. These are called associative arrays as you are associating a key with a value. You can then use the key to access the values.

You use the 'double' array like this:

$person = array(
	'firstname'	=> 'John',
	'lastname'	=> 'Doe',
	'gender'	=> 'male',
	'age'		=> 46,
	'job'		=> 'builder'
	);

You can then access the values using the key within square brackets:

echo $person['firstname']; // John
echo $person['age']; // 46

Changing values

You can update or add new values with the assignment operator (the equals sign), just as you do with other variables.

$person['job'] = 'car salesman' // updates existing value
$person['fav_colour'] = 'blue'; // creates new item

Multi-dimensional arrays

You can have arrays within arrays - so called multi-dimensional arrays.

$person = array(
	'firstname'	=> 'Jane',
	'job'		=> 'full-time mum',
	'children'	=> array( 'Ben', 'Sally', 'Rebecca'),
);

echo $person['children'][0]; // Ben

You can have as many arrays within arrays as you want, though it can get unwieldly quickly.

$person = array(
	'firstname'	=> 'Jane',
	'job'		=> 'full-time mum',
	'children'	=> array(
		array( 'name' => 'Ben', 'age' => 7),
		array( 'name' => 'Sally', 'age' => 9),
		array( 'name' => 'Rebecca', 'age' => 11)
	)
);

We can now loop through each of the children and echo out their details:

foreach ( $person['children'] as $child ){
	echo "{$child['name']} is {$child['age']}";
}
// Ben is 7
// Sally is 9
// Rebecca is 11
