Input Types

There are several different form controls:

Labels

Each form control can have a label associated with it. You can either wrap the control in the label or use the 'for' attribute

<label><input type="checkbox"> Click me</label>
<input type="checkbox" id="clickmetoo">
<label for="clickmetoo">Click me too</label>

Input

There are several different types of inputs, each with their own features. The most common, and default type, is text.

Text

<input type="text" value="Your name" />

There are several attributes you can use, such as title, required, id, name and placeholder.

<form onSubmit="alert('submitted!');return false">
<input type="text" required placeholder="Your name" title="please enter your name" name="name" id="name">
</form>

Hidden

Hidden inputs will appear in your PHP as variables but aren't usually edited by the user. They're a good way of passing variables from page to page.

<input type="hidden" name="secret_var" value="secret">

Password

Password fields work largely like text inputs but will convert input to bullets

<input type="password" value="123">

Submit

These are buttons used to submit a form

<input type="submit" value="Send Form">

Reset

Reset buttons work a lot like submit but they will reset the form to its default values if any were set.

<input type="reset" value="Clear Form">

Checkbox

These will let you check any number of items.

<label><input type="checkbox" value="cats" name="catperson"> I am a cat person</label>
<label><input type="checkbox" value="dogs" name="dogperson"> I am a dog person</label>

Buttons

Buttons are clickable boxes, which you can assign javascript actions to. They can be used to submit forms but are generally used for such things as video controls, etc. You can put images within buttons, though browser styling can be difficult.

<button onClick="alert('click!')" class="btn">Click me!</button>

Radio

Radio buttons will let you select only one of a group of options. It uses the 'name' attribute to make groups.

In this example we've used a label to wrap each radio button. Though this isn't necessary it is a good idea.

<label><input type="radio" name="gender" value="male" /> Male</label>
<label><input type="radio" name="gender" value="female" /> Female</label>
<label><input type="radio" name="gender" value="other" /> Other</label>

Textarea

Textareas are the larger text boxes used to enter longer blocks of text

<textarea name="your_message" id="" cols="10" rows="10">Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod
tempor incididunt ut labore et dolore magna aliqua.</textarea>

Fieldset

These allow you separate your forms into sections. Valid HTML requires form inputs to be within block level elements and fieldsets are a great way of doing this.

You can also use them in conjunction with legend elements to make headings:

<form>
	<fieldset>
		<legend>My Form</legend>
		<input type="text" placeholder="your name">
		<input type="submit" class="btn">
	</fieldset>
</form>

Select

<select name="country">
	<option value="nz" selected>New Zealand</option>
	<option value="aus">Australia</option>
	<option value="usa">U.S.A</option>
	<option value="gb">Great Britain</option>
	<option value="in">India</option>
	<option value="jp">Japan</option>
</select>

You can let users select more options by including the multiple attribute.

<select name="country" multiple="multiple">
	<option value="nz" selected>New Zealand</option>
	<option value="aus">Australia</option>
	<option value="usa">U.S.A</option>
	<option value="gb">Great Britain</option>
	<option value="in" selected>India</option>
	<option value="jp">Japan</option>
</select>
