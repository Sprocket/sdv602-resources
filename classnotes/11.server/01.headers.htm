Headers

PHP is closely connected with HTTP - Hyper-text Transfer Protocol. This is the set of protocols used when web browsers connect to web servers.

Headers

HTTP headers are little chunks of data sent between the server and client. They're used to help control things like authentication, caches, and defining what the web browser will accept.

You can send headers with PHP using the header() function.

Location Header

A very user header to send is location. This will redirect the browser to the specified location. No code after this header will be executed.

header('Location: logout.php');
