HTML

HyperText Markup Language is used to give web pages structure.

It is based around tags, which usually wrap the content. Most of the time they have opening and closing pairs.

<h1>This is a heading.</h1>
<p>This is a paragraph.</p>

Tags can contain other tags, like this:

<p>This is a paragraph with a <a href="home.html">link</a> inside it.</p>

Tags can also have attributes; in this example we're giving the paragraph a class of 'description'

<p class="description">This is a discription.</p>

There's about 100 HTML tags but you'll really only use a subset of 30 or 40. Some common tags include:

<h1>A level one heading</h1>
<title>A page title</title>
<label>A label, used in forms</label>
<div>A div, basically a box</div>

Some tags do not have a closing pair:

<hr>A horizonal rule (a line)
<br>A line break

This will sometimes have a closing slash built into the tag: <br />

And some require other elements to make them useful: like lists, which have a wrapping list tag and then list items:

<ol>
    <li>Item One</li>
    <li>Item Two</li>
    <li>Item Three</li>
</ol>

Ol is an 'ordered list' and the web browser will automatically put in numbers. An 'ul' is an unordered list, and the browser will use bullets.

Attributes

Some tags have attributes to make them more useful. They generally appear as name-value pairs, separated by "=", and are written within the start tag of an element.

<tag attribute="value">{content to be modified by the tag}</tag>

Where tag names the HTML element, attribute is the name of the attribute, set to the provided value.

In this example a link (called an 'anchor') tag needs an href to point to another location, and an image tag needs a src for the image file.

<a href="home.html">Link to home</a>
<img src="logo.png" />

Some attributes are boolean and do not need a value:

<option selected>New Zealand</option>

Page Structure

A well-formed web page will have an html tag, a head tag and a body tag, like this:

<!doctype html>
<html>
    <head>
        <title>My web page</title>
    </head>
    <body>
        <h1>My web page</h1>
    </body>
</html>

The head element doesn't contain things that are shown on the web page, but things like a page title (shown in the title bar), links to style and javascript files, metadata, etc.
