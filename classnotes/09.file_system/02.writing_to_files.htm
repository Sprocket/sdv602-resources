Writing to Files

Touch

This will create a file if it does not exist. If the file does, it will update its modification date.

touch('log.txt');

file_get_contents

Reads entire file into a string and returns the string

$file = file_get_content('data.txt');
echo $file;

file_put_contents

Writes a string to a file. The file is created if it doesn't exist already. It accepts the filename and the content you wish to write to it. It will overwrite any existing content.

$entry = "Dear Diary, today was fun.";
$today = date('Y_m_j') . '.txt'; // 2012_08_14.txt
if ( ! file_put_contents( $today, $entry ) ) {
	echo "I couldn't write today's entry sorry.";
}
