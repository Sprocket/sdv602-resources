Working with CSV Files

Comma separated value files are plain text files that have a list of values separated by commas. Excel will typically render csv files as a spreadsheet. You can also save to them from many word processing and spreadsheet applications as well as scripting languages.

// an example of csv
one,two,three,four

str_getcsv

Parse a CSV string into an array. We pass it a string of csv and it will return an array

$numbers_csv = "one,two,three,four";
$numbers = str_getcsv( $numbers_csv );

print_r($numbers);
[0] => one
[1] => two
[2] => three
[3] => four

You can extract the contents of a csv file using file_get_contents

// notice that the csv file doesn't require the the .csv extension
$words = str_getcsv( file_get_contents('inc/words.txt') );
