Control Structures

Conditional tests are a good way to control the flow of your application. The most commonly used test is the if construct.

If does a test and, if the test is successful (that is, if it returns a truthy value) will then execute the code within curly braces.

if ( test_something ) {
    // do something
}

Sometimes you'll want to do something when the test fails.

if ( test_something ) {
    // do something
} else {
    // do something else
}

For example

if ( $month == 'January' ) {
    echo 'It is January';
} else {
    echo 'It is not January';
}

Other times you will want to run multiple tests:

For example

if ( $month == 'January' ) {
    echo 'It is January';
} elseif ( $month == 'February' )  {
    echo 'It is February';
} else {
    echo 'It is not January or February';
}

You can use as many elseifs as needed, though it will likely make some nasty looking code.

Numbers

You can use some other tests when working with numbers

> greater than
< less than
>= greater than or equal to
<= less than or equal to
!= does not equal (is not)

Functions

You can use the result of functions in your tests too:

if ( isset( $variable )) {
    // variable has been set
}

And also test negation (notice the exclaimation mark)

if ( ! isset( $variable )) {
    // variable has NOT been set
}

OR and AND

You can run multiple tests within the same if. This will test if the month is Janurary or February. It uses the 'double pipe' operator.

if ( $month == 'January' || $month == 'February' ) {
    echo 'It is January or February';
} else {
    echo 'It is not January or February';
}

Another option is to check for two requirements using the AND test - which is two ampersands.

if ( $number >= 0 && $number <= 100 ) {
    echo 'The number is between 1 and 100';
} elseif ( $number > 100 ) {
    echo 'The number is greater than 100';
} else {
    echo 'It is a negative number. How tricky.';
}

Note, you can place each part of the test in separate brackets. You can also substitute && for AND, and || for OR:

if ( ($number >= 1) AND ($number <= 100) ) {
    ...

You can remove the braces if each clause only have a single statement.

if ( $number >= 0 && $number <= 100 ) echo 'The number is between 1 and 100';
elseif ( $number > 100 ) echo 'The number is greater than 100';
else echo 'It is a negative number. How tricky.';
