String Functions

Changing Case

strtoupper

Make a string uppercase

$my_string = 'roses are red';
echo strtoupper($my_string);
// ROSES ARE RED

There are several other similar functions:

strtolower()
Make a string lowercase
ucfirst()
Make a string's first character uppercase
ucwords()
Uppercase the first character of each word in a string
$my_string = 'rOsES Are reD';

echo ucwords( $my_string );
// ROsES Are ReD

echo ucwords( strtolower( $my_string ) );
// Roses Are Red

Trimming

'Trimming' refers to removing whitespace from strings. Whitespace are characters include things like spaces, line returns and tabs.

$user_input = "   I   am 
				a crazy
					    string
			";
echo trim( $user_input );
// I am a crazy string
trim
Strip whitespace (or other characters) from the beginning and end of a string
ltrim
Strip whitespace (or other characters) from the beginning of a string
rtrim
Strip whitespace (or other characters) from the end of a string

printf

One way to format strings is to use the printf function.

$template = "My favourite pet is a %s, her name is %s";
printf( $template, 'cat', 'Samantha');

This will print "My favourite pet is a cat, her name is Samantha".

The template can contain any number of placeholders. They are matched in order with passed values. There must be the same number of values as placeholders.

The placeholders can be of different types and are denoted by a percent sign.

SpecifierMeaning
%sString
%dDecimal
%fFloat

You can use HTML tags within the template

One of the more powerful features is formatting output. Using the template of "%.2f" will give us this:

printf('%.2f', 1); // 1.00
InputOutput
11.00
.90.90
12341234.00

The result will be printed immediately to the browser. If you want to store the result in a variable use the related function sprintf
