Strings

String Functions

strtoupper

Make a string uppercase

$my_string = 'roses are red';
echo strtoupper($my_string);
// ROSES ARE RED

There are several other similar functions:

strtolower()
Make a string lowercase
ucfirst()
Make a string's first character uppercase
ucwords()
Uppercase the first character of each word in a string
$my_string = 'rOsES Are reD';

echo ucwords( $my_string );
// ROsES Are ReD

echo ucwords( strtolower( $my_string ) );
// Roses Are Red

Entities

htmlentities

Convert all applicable characters to HTML entities

$html = "<h1>I am some HTML</h1>";
echo htmlentities( $html );
// &lt;h1&gt;I am some HTML&lt;/h1&gt;

urlencode

This function is convenient when encoding a string to be used in a query part of a URL, as a convenient way to pass variables to the next page.

$link = 'my super gallery of holiday photos!';
echo '<a href="/gallery.php?gallery_name='
	. urlencode( $link )
	. '">Next Gallery</a>';
// <a href="/gallery.php?gallery_name=my+super+gallery+of+holiday+photos%21">Next Gallery</a>

Trimming

'Trimming' refers to removing whitespace from strings. Whitespace are characters include things like spaces, line returns and tabs.

$user_input = "   I   am 
				a crazy
					    string
			";
echo trim( $user_input );
// I am a crazy string
trim
Strip whitespace (or other characters) from the beginning and end of a string
ltrim
Strip whitespace (or other characters) from the beginning of a string
rtrim
Strip whitespace (or other characters) from the end of a string
