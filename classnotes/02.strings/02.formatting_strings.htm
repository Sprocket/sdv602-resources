Formatting Strings

printf

One way to format strings is to use the printf function.

$template = "My favourite pet is a %s, her name is %s";
printf( $template, 'cat', 'Samantha');

This will print "My favourite pet is a cat, her name is Samantha".

The template can contain any number of placeholders. They are matched in order with passed values. There must be the same number of values as placeholders.

The placeholders can be of different types and are denoted by a percent sign.

SpecifierMeaning
%sString
%dDecimal
%fFloat

You can use HTML tags within the template

One of the more powerful features is formatting output. Using the template of "%.2f" will give us this:

printf('%.2f', 1); // 1.00
InputOutput
11.00
.90.90
12341234.00

The result will be printed immediately to the browser. If you want to store the result in a variable use the related function sprintf
