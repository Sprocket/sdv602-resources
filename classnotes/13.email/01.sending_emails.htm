Sending Emails with PHP

PHP has a built in function to send emails.

Note you will need to have email set up on your web server or use a third party.

bool mail ( string $to , string $subject , string $message [, string $additional_headers [, string $additional_parameters ]] )

mail('user@website.co.nz', 'test email', 'this is a test email');

You can also provide more details using headers.

if ( send_test_email() ) echo 'mail sent!';

function send_test_email(){

	$to      = 'Bill Gates <billg@microsoft.com>';
	$subject = 'test email message';
	$message = 'testing!';
	$headers = 'From: Me <webmaster@website.com>' . "\r\n" .
	    'Reply-To: webmaster@website.com' . "\r\n" .
	    'X-Mailer: PHP/' . phpversion();

	return mail($to, $subject, $message, $headers);
}

if ( send_test_email() ) echo 'mail sent!';
else echo 'nope, soz';

You can pass HTML as the content as well as a plain-text version.
