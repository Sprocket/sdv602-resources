Input Types

There are several different form controls:

Textarea is a large text box for users to enter in chunks of text
Button is a clickable box, which you can assign javascript actions to
Select is a pop up menu

There are also variations of the input type including:

Hidden that the user cannot edit
Submit a submit button
Reset that will reset the from to its initial state
Checkbox is a binary control allow the user to choose between two options

Some controls have several options, such as pop up select menus and radio groups.

<select name="country">
	<option value="nz" selected>New Zealand</option>
	<option value="aus">Australia</option>
	<option value="usa">U.S.A</option>
</select>
