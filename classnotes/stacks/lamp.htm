The LAMP Stack

A popular architecture for deploying web applications:

Linux Operating System

Apache Web Server

MySQL Database

PHP Scripting Language

(The 'P' can also include Python and Perl)

Because PHP is cross-platform the operating can also be swapped out for Macintosh, Windows or Other: MAMP, WAMP and XAMP, respectively. As of 5.4 it also has a built-in web server (not recommended for production).

PHP can also run quite happily on IIS, and Nginx, and many other HTTP servers.

It also supports a wide range of databases * MySQL * PostgreSQL * SQL Server * Oracle * SQLite * ODBC, and more

PHP 5.1 supports PDO (PHP Data Objects) interface for accessing databases. Each database requires a driver but will then allow you to access the data in a convenient way.
