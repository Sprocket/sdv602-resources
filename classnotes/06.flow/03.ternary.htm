Ternary Operator

The conditional operator (a ternary operator) performs a test and evaluates either the second or third expression depending on the evaluation of the first expression.

A question mark separates the test from the possible outcomes, which are separated by a colon. Parenthesis around the test are optional but reccomended.

( test ) ? TRUE : FALSE;

It can be used to assign a value to a variable:

$gender = 'boy';
$name = ( $gender == 'boy' ) ? 'Sam' : 'Sue';
// Sam

Or to echo something to the browser:

$number = 2;
echo "$number is ";
echo ( $number > 0 ) ? 'positive' : 'negative';
// 2 is positive

Or call a function, here we're formatting it over multiple lines for readability:

test_something()
? do_something_on_true()
: do_something_on_false();
