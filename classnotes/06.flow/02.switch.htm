Switch statements

Switch statements are similar to IF statements but can be a more elegant solution to making multiple tests.

The are limited to a single test but tend to be a better solution to multiple IF and IF ELSE clauses.

$fav_fruit = "apple";

switch ( $fav_fruit ) {
	
	case 'banana' :
		echo 'Monkies like bananas too!';
		break;

	case 'grapes' :
		echo 'Mmmm, grapes';
		break;

	case 'apple' :
		echo 'Yay, apples are tasty!';
		break;

	default :
		echo 'You like some strange food my friend';

}
// echos 'Yay, apples are tasty!'

Break statements stop control from cascading down the structure. When a case is truthy all code below it will be executed. This can be sometimes be useful:


$fav_tvshow = 'MacGyver';

switch ( $fav_tvshow) {
	case 'The A Team':
	case 'MacGyver':
	case 'KnightRider':
	case 'That\'s Incredible':
		echo 'That is a great show.';
		break;
	
	default:
		echo 'Yeah, not one of my favourites.';
		break;
}
// echos 'That is a great show.'
